frontmatter `tags` to notes generated by `playlist-to-brain`. Bias toward this
vocabulary; only introduce a new tag when nothing here fits.

Edit freely. Delete entries you'll never use; add ones that match how you
already think. If you delete this file entirely, the agent will pick tags from
scratch per video.
